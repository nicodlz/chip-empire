These features are **non-negotiable**. Every successful idle game has them.

### 1. Core Click Loop
- **Manual clicking** as the initial interaction (1 resource per click to start)
- Immediate, satisfying feedback on every click
- Visual/audio feedback for clicking (numbers flying, sounds, particles)
- Cookie Clicker: Click cookie â†’ get cookie. Universally understood.

### 2. Automation (Idle Production)
- **Passive income generators** that work without clicking
- Production continues when the app is closed (offline progress)
- Multiple tiers of automation (cursors â†’ grandmas â†’ factories)
- AdVenture Capitalist: Managers run businesses automatically

### 3. Upgrades System
- **Permanent upgrades** that multiply production
- Clear progression path with visible next goals
- Upgrades for both active (click power) and passive (idle speed)
- Exponential cost scaling (typically 15-20% per purchase)

### 4. Prestige/Reset Mechanic
- **Soft reset** that trades progress for permanent bonuses
- "Prestige currency" (Hero Souls, Heavenly Chips, Angel Investors)
- Makes the game infinitely replayable
- Universal Paperclips: New Universe option after completion
- Cookie Clicker: Ascension with Heavenly Chips (+1% per level)

### 5. Number Visualization
- **Big number notation** (scientific, suffixes like M, B, T, Qa, Qi...)
- Satisfying number growth animations
- Clear display of production rates (per second/minute)
- Antimatter Dimensions: Numbers reaching 10^308 and beyond

### 6. Offline Progress
- **Accumulation while away** (essential for mobile)
- "Welcome back! You earned X while away"
- Reasonable caps or diminishing returns to encourage return visits

### 7. Save System
- **Auto-save** frequently
- Cloud save / cross-device sync (for mobile)
- Export/import saves (hardcore players demand this)

### 8. Clear Goals & Milestones
- **Visible next unlock** always on screen
- Progress bars toward next achievement/upgrade
- "You need X more to unlock Y"
