# Roadmap: Chip Empire

## Overview

Build a semiconductor empire idle game. Mine real minerals, fab chips at shrinking nodes, accumulate compute power, unlock research. Mobile-first, satisfying, modern.

## Phases

- [ ] **Phase 1: Foundation** — Tap to mine, base UI, Tier 1 minerals
- [ ] **Phase 2: Fabrication** — Wafers, chips, 90nm process, save/load
- [ ] **Phase 3: Compute** — FLOPS generation, research tree, node progression
- [ ] **Phase 4: Scale** — Automation, offline progress, Tier 3-4 minerals
- [ ] **Phase 5: Endgame** — Advanced nodes (7nm-3nm), final research, polish

## Phase Details

### Phase 1: Foundation

**Goal**: Tap to mine minerals with satisfying feedback, modern dark UI
**Depends on**: Nothing
**Requirements**: CORE-01, CORE-02, CORE-03, MIN-01, UI-01, UI-03, UI-04
**Success Criteria**:
1. Tap anywhere → gain Silicium
2. Number flies up with animation
3. Resource panel shows Silicium, Cuivre, Aluminium
4. Dark theme with neon accents
5. Works smooth on mobile
6. 60fps animations

**Plans**: 3

### Phase 2: Fabrication

**Goal**: Turn minerals into chips, first taste of progression
**Depends on**: Phase 1
**Requirements**: FAB-01, FAB-02, FAB-03, CORE-04, MIN-02, UI-05
**Success Criteria**:
1. Combine minerals → Wafer
2. Wafer → CPU chip (90nm)
3. Chip appears in inventory
4. Save/load works
5. Tier 2 minerals unlockable
6. Fab panel with costs

**Plans**: TBD

### Phase 3: Compute

**Goal**: Chips generate FLOPS, unlock research
**Depends on**: Phase 2
**Requirements**: CALC-01, CALC-02, CALC-03, RES-01, RES-02, RES-03, FAB-04, MIN-03
**Success Criteria**:
1. Chips generate FLOPS over time
2. FLOPS counter (TFLOPS, PFLOPS...)
3. Research tree visible
4. Can buy research with FLOPS
5. Research unlocks 65nm → 45nm → 28nm
6. Tier 3 minerals via research

**Plans**: TBD

### Phase 4: Scale

**Goal**: Automation, play while away
**Depends on**: Phase 3
**Requirements**: AUTO-01, AUTO-02, AUTO-03, AUTO-04, OFF-01, OFF-02, ACH-01, MIN-04
**Success Criteria**:
1. Auto-miners purchasable
2. Auto-fab chains
3. Datacenters generate FLOPS
4. Offline progress calculated
5. "Welcome back" modal
6. Basic achievements

**Plans**: TBD

### Phase 5: Endgame

**Goal**: Race to 3nm, final research, polish
**Depends on**: Phase 4
**Requirements**: FAB-05, RES-04, RES-05, MIN-05, ACH-02, UI-06
**Success Criteria**:
1. 14nm → 7nm → 5nm → 3nm unlockable
2. Endgame research (IA, Quantique, Fusion)
3. Tier 5 ultra-rare minerals
4. Achievements give bonuses
5. Game feels complete
6. No clutter, polished UX
