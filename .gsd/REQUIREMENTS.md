# Requirements: Chip Empire

## v1 Requirements

### Core Loop
- [ ] **CORE-01**: Tap to mine (satisfying feedback)
- [ ] **CORE-02**: Visual feedback on tap (number + particles)
- [ ] **CORE-03**: Resource counters update real-time
- [ ] **CORE-04**: Save/load (localStorage)

### Minerais (15+ types)
- [ ] **MIN-01**: Tier 1 — Silicium, Cuivre, Aluminium (communs)
- [ ] **MIN-02**: Tier 2 — Or, Argent, Étain (métaux)
- [ ] **MIN-03**: Tier 3 — Lithium, Cobalt, Tantale (terres rares)
- [ ] **MIN-04**: Tier 4 — Néodyme, Gallium, Indium (exotiques)
- [ ] **MIN-05**: Tier 5 — Germanium, Hafnium (ultra-rares, late game)
- [ ] **MIN-06**: Chaque minerai a des propriétés (rareté, usage)
- [ ] **MIN-07**: Unlock progressif des tiers

### Fabrication de Puces
- [ ] **FAB-01**: Combiner minerais → Wafer
- [ ] **FAB-02**: Wafer → Puce (CPU, GPU, Mémoire)
- [ ] **FAB-03**: Procédé 90nm (starter)
- [ ] **FAB-04**: Procédés 65nm → 45nm → 28nm
- [ ] **FAB-05**: Procédés 14nm → 7nm → 5nm → 3nm
- [ ] **FAB-06**: Chaque node = minerais + recherche spécifiques

### Puissance de Calcul
- [ ] **CALC-01**: Puces génèrent FLOPS (passif)
- [ ] **CALC-02**: FLOPS = monnaie pour recherche
- [ ] **CALC-03**: Affichage en notation scientifique (TFLOPS, PFLOPS, EFLOPS)

### Recherches
- [ ] **RES-01**: Arbre de recherche
- [ ] **RES-02**: Recherches débloquent minerais
- [ ] **RES-03**: Recherches débloquent procédés
- [ ] **RES-04**: Recherches débloquent automatisation
- [ ] **RES-05**: Milestones: IA, Quantique, Fusion

### Automatisation
- [ ] **AUTO-01**: Mines automatiques (par minerai)
- [ ] **AUTO-02**: Chaînes de fabrication auto
- [ ] **AUTO-03**: Datacenters (FLOPS passifs)
- [ ] **AUTO-04**: Upgrades d'efficacité

### Offline & Retention
- [ ] **OFF-01**: Calcul gains offline au retour
- [ ] **OFF-02**: "Welcome back" avec résumé
- [ ] **ACH-01**: Achievements/milestones
- [ ] **ACH-02**: Achievements donnent bonus

### UI/UX
- [ ] **UI-01**: Mobile-first responsive
- [ ] **UI-02**: Touch targets ≥ 44px
- [ ] **UI-03**: Design moderne épuré
- [ ] **UI-04**: Dark theme + accents néon
- [ ] **UI-05**: Animations fluides (60fps)
- [ ] **UI-06**: Pas de clutter, clarté max

## v2 Requirements (Deferred)

- [ ] Prestige system ("Nouvelle ère")
- [ ] Sound effects
- [ ] Events spéciaux
- [ ] Cloud save
- [ ] Plus de types de puces (NPU, TPU)
- [ ] Marché des minerais (fluctuations)

## Out of Scope

- Multiplayer — v1 solo only
- Microtransactions — pas le but
- Backend — tout local
- Simulation économique réaliste
